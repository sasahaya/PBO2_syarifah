Petshop Inventory - AdminLTE Laravel Template

Instructions:
1. Copy these files into a Laravel project's root.
2. Run composer install and npm install if you use frontend assets.
3. Set up .env and database.
4. Run migrations: php artisan migrate
5. Serve: php artisan serve

Notes:
- This template uses AdminLTE via CDN for simplicity.
- Add auth middleware as needed.
